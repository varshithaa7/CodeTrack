# CodeTrack
The Unified Coding Progress Tracker is a platform designed to help developers and learners track their progress across multiple coding platforms such as LeetCode, GitHub, Codeforces, HackerRank, and GeeksforGeeks. The website consolidates achievements, coding statistics, and hackathon participation into a single dashboard, making it easier for users to monitor their growth and identify areas for improvement.
